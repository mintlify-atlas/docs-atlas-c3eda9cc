## Interface Definition

The `Tratamiento` interface represents a treatment or dental service available in the clinic.

```typescript
export interface Tratamiento {
  id: number;
  nombre: string;
  categoria: string;
  descripcion: string;
  duracion: number;
  precio: number;
}
```

## Properties

<ResponseField name="id" type="number" required>
  Unique identifier for the treatment
</ResponseField>

<ResponseField name="nombre" type="string" required>
  Treatment name (e.g., "Limpieza dental", "Ortodoncia", "Implante dental")
</ResponseField>

<ResponseField name="categoria" type="string" required>
  Treatment category. Common values include:
  - `Preventiva`: Preventive care
  - `Ortodoncia`: Orthodontics
  - `Estética`: Cosmetic dentistry
  - `Cirugía`: Surgery
</ResponseField>

<ResponseField name="descripcion" type="string" required>
  Detailed description of the treatment procedure
</ResponseField>

<ResponseField name="duracion" type="number" required>
  Expected duration in minutes
</ResponseField>

<ResponseField name="precio" type="number" required>
  Treatment cost in euros (€)
</ResponseField>

## Example

```json
{
  "id": 1,
  "nombre": "Limpieza dental",
  "categoria": "Preventiva",
  "descripcion": "Limpieza profesional y eliminación de sarro.",
  "duracion": 45,
  "precio": 80
}
```
